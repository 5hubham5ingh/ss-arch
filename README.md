# Arch-Based Desktop Environment  

- **Hyprland:** Wayland compositor and window manager. [ðŸ”—](https://hyprland.org/)  
- **Kitty:** Terminal emulator.   [ðŸ”—](https://sw.kovidgoyal.net/kitty/)  
- **Jiffy:** Application launcher, window switcher, emoji picker, basic calculator, Hyprland's keybinds browser, and custom command menus (e.g., power menu, screenshot capture).  [ðŸ”—](https://github.com/5hubham5ingh/jiffy)   
- **Kitty Panel:** Terminal-based system panel for real-time system monitoring.   [ðŸ”—](https://github.com/5hubham5ingh/kitty-panel)  
- **VSCode:** IDE.   [ðŸ”—](https://code.visualstudio.com/)  
- **NVIM:** Text editor.   [ðŸ”—](https://neovim.io/)  
- **Lazygit:** Version control.   [ðŸ”—](https://github.com/jesseduffield/lazygit)  
- **Foxpanel:** GUI system panel for system control and utilities (screen brightness control, system volume control, WiFi and Bluetooth connection management, etc.), plus various web-based widgets (screen recorder, wallpaper browser, music player, weather updates, calendar, etc.).   [ðŸ”—](https://github.com/5hubham5ingh/foxpanel)  
- **WallRizz:** System wallpaper and theme manager.   [ðŸ”—](https://github.com/5hubham5ingh/WallRizz)  

There are currently three different system themes and configuration here which can be change dynamically.
